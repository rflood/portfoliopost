Sanctuary

Website produced via custom WordPress theme for a proposal for an artists retreat.

Produced: 2014

The Sanctuary site was produced as a mock-up. The business proposal is for a artists retreat in the Wisconsin woodlands that features cabins specific to individual artistic disciplines, community events, and organic cuisine. The site was developed via a content management system, so that content could hypothetically be maintained by the client. I produced a custom Wordpress theme to present the content, which included a scrolling homepage, and a single cabin selection featuring a gallery slideshow. This project employed the use of HTML, CSS, PHP, Javascript, JQuery, and the WordPress CMS.
